# KiraBiz Accounting Plus - Marketing Site

This folder contains the public-facing landing page and documentation for KiraBiz Accounting Plus.

## Contents
- `index.html`: Main landing page (Tailwind CSS).
- `guide.html`: Quick Start Guide for users (Bootstrap).
- `images/`: High-resolution logos (LHDN, PERKESO, KWSP) and application icons.
- `favicon.ico`: Application favicon.

## Deployment
This site is deployed to GitHub Pages at:
https://axesude.github.io/kirabiz-site/

To update the site, push changes from this folder to the `main` branch of the repository.

## Features Highlighted
- Malaysian Statutory Compliance (EPF, SOCSO, EIS, PCB).
- SST (Sales and Service Tax) Automation.
- Modern React UI.
- Local-first data privacy.
- Windows x86, x64, and ARM64 support.
